# 📂 Практика: Внутрішні посилання HTML
### Покроковий гайд — «Сонячна система»

---

## 🎯 Що ти відпрацюєш

- Внутрішні посилання між HTML-файлами (`<a href="...">`)
- Відносні шляхи: до підпапки, до батьківської папки, через кілька рівнів
- Якірні посилання (`<a name="...">` та `href="#..."`)
- Структуру сайту з трьома рівнями вкладеності

---

## 📁 Структура файлів

Перш ніж відкривати редактор — **створи таку структуру папок і файлів**:

```
solar_system/
│
├── index.html               ← 🌟 Рівень 1: Головна
├── about.html               ← 🌟 Рівень 1: Про проект
│
├── inner/                   ← папка «Внутрішні планети»
│   ├── index.html           ← 🔴 Рівень 2: Список внутрішніх планет
│   ├── mercury.html         ← 🔴 Рівень 2: Меркурій
│   ├── venus.html           ← 🔴 Рівень 2: Венера
│   │
│   └── earth/               ← папка «Земля»
│       ├── index.html       ← 🟢 Рівень 3: Земля
│       └── moon.html        ← 🟢 Рівень 3: Місяць
│
└── outer/                   ← папка «Зовнішні планети»
    ├── index.html           ← 🟣 Рівень 2: Список зовнішніх планет
    ├── jupiter.html         ← 🟣 Рівень 2: Юпітер
    │
    └── saturn/              ← папка «Сатурн»
        ├── index.html       ← 🟡 Рівень 3: Сатурн
        └── rings.html       ← 🟡 Рівень 3: Кільця Сатурна
```

**Усього:** 10 HTML-файлів, 4 папки, 3 рівні вкладеності.

---

## 🧭 Головне правило відносних шляхів

| Ти знаходишся в... | Хочеш дістатися до... | Пишеш у `href` |
|---|---|---|
| `solar_system/` | `inner/index.html` | `inner/index.html` |
| `inner/` | `../index.html` (головна) | `../index.html` |
| `inner/` | `../outer/index.html` | `../outer/index.html` |
| `inner/earth/` | `../../index.html` (головна) | `../../index.html` |
| `inner/earth/` | `../index.html` (inner) | `../index.html` |
| `inner/earth/` | `moon.html` (та сама папка) | `moon.html` |
| `outer/saturn/` | `../../inner/earth/index.html` | `../../inner/earth/index.html` |

**Запам'ятай:**
- `../` — піднятися на **один рівень** вгору
- `../../` — піднятися на **два рівні** вгору
- `назва_папки/` — зайти **всередину** папки
- просто `файл.html` — файл **у тій самій папці**

### Крок 2 — Шаблон кожної сторінки

Кожен HTML-файл має таку базову структуру. **Відкрий будь-який редактор
та скопіюй шаблон:**

```html
<!doctype html>
<html lang="uk">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Назва сторінки</title>
  </head>
  <body style="font-family: 'Segoe UI', sans-serif; margin: 0;">

    <!-- НАВІГАЦІЯ -->
    <p style="background-color: #1a1a2e; color: white; padding: 12px 20px; margin: 0;">
      <a href="ШЛЯ_ДО_ГОЛОВНОЇ"  style="color: white; text-decoration: none;">🌟 Головна</a>  |
      <a href="ШЛЯХ_ДО_INNER"    style="color: white; text-decoration: none;">🔴 Внутрішні</a>  |
      <a href="ШЛЯХ_ДО_OUTER"    style="color: white; text-decoration: none;">🪐 Зовнішні</a>
    </p>

    <!-- КОНТЕНТ -->
    <div style="width: 700px; margin: auto;">
      <a name="top"></a>
      <h1>Назва сторінки</h1>
      <hr />

      <p>Текст сторінки...</p>

      <!-- ЯКІР "ВГОРУ" -->
      <p style="text-align: center;">
        <small><a href="#top">↑ На початок</a></small>
      </p>
    </div>

  </body>
</html>
```

### Крок 3 — `solar_system/index.html` (Рівень 1 — Головна)

**Де знаходиться файл:** `solar_system/index.html`

Особливість: це **кореневий файл**, тому всі шляхи ведуть лише
вглиб — в підпапки.

```html
<!-- Навігація — ти вже на головній, посилання на інші розділи: -->
<a href="inner/index.html">🔴 Внутрішні планети</a>
<a href="outer/index.html">🪐 Зовнішні планети</a>
<a href="about.html">📖 Про проект</a>
```

Що написати у `<body>`:
- Заголовок `<h1>` — «Сонячна система»
- Два параграфи з описом
- Підзаголовок `<h2>` для внутрішніх планет + посилання на `inner/index.html`
- Підзаголовок `<h2>` для зовнішніх планет + посилання на `outer/index.html`
- Використай `<strong>`, `<em>`, `<span style="color:...">` для оформлення

### Крок 4 — `solar_system/inner/index.html` (Рівень 2)

**Де знаходиться файл:** `inner/index.html`

Ти тепер **всередині папки `inner/`**, тому щоб дістатися до
кореневого `index.html` — потрібно вийти на рівень вгору: `../`

```html
<!-- Навігація з папки inner/: -->
<a href="../index.html">🌟 Головна</a>
<a href="../outer/index.html">🪐 Зовнішні планети</a>
```

Що написати у `<body>`:
- Список планет: Меркурій, Венера, Земля, Марс
- Кожна назва — це посилання на відповідну сторінку:
  ```html
  <a href="mercury.html">☿ Меркурій</a>
  <a href="venus.html">♀ Венера</a>
  <a href="earth/index.html">🌍 Земля</a>
  ```
- Зверни увагу: `mercury.html` і `venus.html` — у **тій самій папці** `inner/`,
  а `earth/index.html` — у **підпапці** `earth/`

### Крок 5 — `solar_system/inner/mercury.html` та `venus.html` (Рівень 2)

Ці два файли знаходяться в `inner/` — тому навігація така сама, як у Кроці 4.

Для Меркурія:
```html
<!-- Посилання "далі/назад" між планетами: -->
← <a href="index.html">Всі внутрішні планети</a>
→ <a href="venus.html">Наступна: Венера</a>
→ <a href="earth/index.html">Земля</a>
```

Що написати у `<body>`:
- Характеристики планети (використай `<br />` для переносу рядків)
- Текст з виділеннями через `<strong>`, `<em>`, `<span>`
- Температуру виділи кольором: `<span style="color: #c0392b;">+430 °C</span>`

### Крок 6 — `solar_system/inner/earth/index.html` (Рівень 3 ⚠️)

**Де знаходиться файл:** `inner/earth/index.html`

Ти тепер на **третьому рівні**! До кореня треба піднятися **двічі**: `../../`

```html
<!-- Навігація з папки inner/earth/: -->
<a href="../../index.html">🌟 Головна</a>           ← два рівні вгору
<a href="../index.html">🔴 Внутрішні планети</a>    ← один рівень вгору
<a href="../../outer/index.html">🪐 Зовнішні</a>    ← два вгору, потім в outer/
```

Посилання всередині сторінки:
```html
<!-- moon.html — у тій самій папці earth/ -->
<a href="moon.html">🌕 Місяць</a>

<!-- Перехрест між розділами: з inner/earth/ до outer/ -->
<a href="../../outer/index.html">Зовнішні планети</a>
```

### Крок 7 — `solar_system/inner/earth/moon.html` (Рівень 3)

Файл знаходиться **поряд з** `inner/earth/index.html` — в тій самій папці.

```html
<!-- Посилання назад до Землі — та сама папка, просто ім'я файлу: -->

<!-- На рівень вище — до inner/: -->
<a href="../index.html">Внутрішні планети</a>

<!-- На два рівні вище — до кореня: -->
<a href="../../index.html">🌟 На головну</a>
```

## 💡 Типові помилки

| Помилка | Причина | Рішення |
|---|---|---|
| Посилання не відкривається (404) | Неправильна кількість `../` | Рахуй: скільки папок між файлом і метою? |
| Посилання відкриває порожню сторінку | Файл ще не створено | Спочатку створи всі файли |
| Сторінка не прокручується на початок | `<a name="top">` відсутній | Додай тег-якір перед заголовком |
| Навігація не збігається на різних сторінках | Копіювали без зміни шляхів | Шляхи треба переписувати для кожного рівня |

## 🎓 Що ти навчився

Після виконання цього завдання ти вмієш:

1. **Будувати структуру** проекту з кількома рівнями папок
2. **Відносні шляхи** — переходи вгору (`../`) і вниз (назва папки)
3. **Якірні посилання** — `<a name="id">` + `<a href="#id">`
4. **Навігацію** — єдина панель посилань, що веде по всьому сайту
5. **Крос-посилання** — між абсолютно різними розділами сайту
